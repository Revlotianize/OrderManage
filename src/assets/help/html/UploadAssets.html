Upload Assets

There are two ways to upload assets, you can either use 'Asset Upload' to select assets and then add the asset metadata or you can run large scale uploads offline using 'Batch Upload', via FTP. You can upload and store any file type.

Asset Upload

Click 'Asset upload', select one or more assets then add metadata and upload them. While uploading more than one assets, you can either add common metadata to all the assets or add unique metadata for each asset individually.


To upload assets using 'Asset Upload' :

    1. Click ‘Upload Asset’  
    2. Browse and select the assets for upload


    1. Click ‘Open’ to upload the selected assets        



    1. The assets display in ‘Upload assets’ page




    1. The number of assets added display in top left  ‘Uploading assets’


Adding metadata: You can add metadata for all the assets at once or you can select a metadata attribute and add it for an individual asset.
