Asset Activity

Asset related operations performed by a user are tracked and detailed in a newsfeed format, allowing you to keep track of all asset activities with the click of a button. User actions that are tracked and recorded include:

    • Upload
    • Delete
    • Download
    • Email
    • Add/edit asset metadata



Activity details for an asset uploaded, viewed, downloaded and shared includes -

      • User:  User who performed the action
      • Action: Details of the action that was performed
      • Timestamp: Date and time of the action



Activity details for edits made in the asset metadata includes -



      • User: user who performed the action
      • Action:  Made changes in metadata
      • Original value:  Previous metadata value
      • New value: Edited metadata value
      • Timestamp: Date and time of the action

       

To view activity:

    1. Click 'Activity'  
    2. Asset activity logs display
