Assets

On successful login, the landing page is the 'Assets' page. The 'Assets' page easily allows you to upload, sort, search, apply filters and navigate assets. You can apply an action from view, edit, download, share and delete on multiple assets or view an asset in detail by selecting it. The various features on the Assets page are explained in detail below.

The 'Assets' page is split into four sections as follows -

  • Title Bar

The title bar consists of -

      • Main Menu expand/collapse
      • Profile
      • Switch apps
      • Search box

  • Task Bar

The task bar consists of -

The task bar allows quick access to features and includes:

      • Upload Assets
      • Sort Assets


      • Bulk Actions

Select one or more assets from the asset browser pane to activate the bulk action panel


      • Filters
      • Predefined default filter


      • Applied Filters
        • Save filters
      • More options (Saved filters list)



  • Main Menu Navigation :

The left pane 'Main Menu' consists of -



      • Assets to easily upload, manage, sort and search assets
      • Batch Upload to run large-scale uploads offline
      • Reports to generate reports for various asset related information
      • Help for step by step instructions to guide you through all the features


  • The Asset Browser Pane :

The Asset Browser  provides an overview of your project's content. You can browse your project in its entirety, search for particular assets sort and navigate through the assets.
