Asset Info

An asset is a representation of your files and can be made up of one or more attachments. You can add  different types of files associated with the asset in one place, and the attached assets are termed as 'Asset Files'. Each asset file can have one or more additional attachments, these files are generally the source relation of an asset file, for example a pdf document,localized versions of the same document, or different view angles of the same product. Additional attachments can be added or removed from an asset by editing from the 'Asset Info' page.

The 'Asset Info' page provides you with the enlarged asset image in the preview pane and its metadata under 'Asset Details'. Metadata is information about the asset and this information consists of a collection of attributes, that help in managing and organizing the asset more efficiently so it can be easily retrieved.  

The metadata consists of two types of attributes, the basic system attributes and the custom attributes :

    • 'System Attributes' are standard attributes applied to all assets and include the following -
      • Asset Code
      • Asset Title
      • Description
      • Tags
    • 'Custom Attributes' are configurable attributes applied to describe the asset.
