Batch Uplaod

When you need to upload large number of assets or assets of very large sizes, then uploading them through the UI is not suitable as it becomes cumbersome and results in waste of time. In such cases, use the ‘Batch Upload’ feature to upload assets on to the server via the FTP mechanism.

Batch Upload allows you to add metadata for all of the assets in a separate template file and upload it along with the asset files folder. A standard metadata template is provided in the ‘Batch Upload’ window and can be downloaded from there.

The metadata template includes the following attributes –

  • Asset Title
  • Asset Code
  • Other attributes


Fill the metadata template and validate it to avoid issues during upload. Once the asset files and the template are added, batch upload process starts. In the process, assets are picked from the specified folder location and corresponding metadata from the template is added to the asset and then published. The time taken to upload all the assets depends on the size of files and your network bandwidth. You can continue working in the portal during the upload process.

On successful completion of the upload process, the metadata template is updated with file upload status and reason for upload failure details for the individual assets.


To 'Batch Upload' assets:

    1. Select ‘Batch Uploads’ from main menu



    1. The Batch Details page displays
    2. Click 'Batch Upload'  
    3. Click the ‘Site’ dropdown and select from options



You can download the batch template, or you can skip the process in case you already have the template downloaded.


To download the template:

    1. Click 'Download and Continue' to download the batch template
    2. Add data to downloaded file
  • Open the downloaded excel file
  • Enter the corresponding data values in subsequent rows

Note: the asset name should match the name in the template exactly, including any spaces and capitalization.


To skip the downloading process:

    • Click  to go to ‘FTP Credentials’ tab


To enter ftp credentials:


    1. Enter the server name in ‘FTP SERVER’ field
    2. Enter the FTP folder name in ‘FTP FOLDER’ field
    3. Enter username in ‘USER NAME’ field
    4. Enter the password in ‘PASSWORD’ field
    5. Click ‘Next’ to go to ‘Start Batch Upload’


To start batch upload:

    1. Upload filled template

You can upload the template in two ways –

      • Click to browse and select the filled template  

Or

      • Drag and drop the filled template to  
    1. Click ‘Start Batch Upload’ to start the process
